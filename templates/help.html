📘 RollCall3 – Help & Instructions

Follow these steps to create events, verify attendance, and request POAP mint links.

📝
Step 1: Create Event

Fill in event details (name, description, city, country, dates, email, and secret code). Upload an event icon (PNG/JPG) to represent your POAP.

📂
Step 2: Upload Attendance Report

Upload the Zoom/CSV/XLS report with attendee data. RollCall3 will automatically verify participant attendance.

✅
Step 3: Verify Event

RollCall3 will confirm the event creation through POAP API. You will see your Event ID and Secret Code.

🚀
Step 4: Request Mint Links

Go to the POAP Dashboard and import your Event ID + Secret Code to request mint links. Once approved, links can be distributed to students.

📩
Step 5: Share POAPs

After approval, RollCall3 can send mint links to students automatically via email or display them as QR codes in the dashboard.
